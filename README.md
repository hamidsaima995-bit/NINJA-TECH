# Ninja Tech

Portfolio site for Ninja Tech — the freelance development practice behind ODDEX VIBE, BiltyVault, and Munshi Jee.

**Live:** [ninja-tech-production.up.railway.app](https://ninja-tech-production.up.railway.app)

---

## About

A single-page portfolio presenting shipped work, the stack behind it, and a way to get in touch. Built to load fast and read clearly on a phone, because that is where most people open a link from a proposal.

## Features

- Project showcase with live links and stack details
- Responsive layout from mobile through desktop
- Contact form
- Fast static build with no framework overhead beyond React

## Tech stack

| Layer | Technology |
|---|---|
| Frontend | React, Vite |
| Styling | CSS |
| Hosting | Railway |

## Running locally

```bash
git clone https://github.com/hamidsaima995-bit/NINJA-TECH.git
cd NINJA-TECH
npm install
npm run dev
```

## Services

Ninja Tech builds web applications for small businesses and startups:

- Business tools — ledgers, inventory, billing, CRM
- Web applications with React and Node.js
- Supabase backends and database design
- AI integrations including chatbots and content generation
